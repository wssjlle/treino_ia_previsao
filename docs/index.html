Descrição

Este projeto realiza análise e classificação de clientes a partir dos arquivos de dados:

  • Carrega `clientes.csv` e `novos_clientes.csv`
  • Pré-processamento e codificação de variáveis
  • Treinamento de modelos KNN e Random Forest
  • Avaliação de acurácia e relatórios de performance

Como Usar

  1. Clone o repositório:
    git clone https://github.com/wssjlle/meu_projeto.git
  2. Entre na pasta:
    cd meu_projeto
  3. Crie e ative um ambiente virtual:
    python -m venv venv && source venv/bin/activate
  4. Instale dependências:
    pip install -r requirements.txt
  5. Execute a aplicação:
    python src/app.py
